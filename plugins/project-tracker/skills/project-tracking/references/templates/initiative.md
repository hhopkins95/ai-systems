# Initiative Template

Use this for creating new `INITIATIVE.md` files.

---

```markdown
---
title: [Initiative Name]
created: [YYYY-MM-DD]
status: active
---

# [Initiative Name]

## Goal

[What are we trying to accomplish? Why does this matter?]

## Scope

**In scope:**
- [Thing 1]
- [Thing 2]

**Out of scope:**
- [Thing explicitly not included]

## Completion Criteria

- [ ] [Criterion 1]
- [ ] [Criterion 2]
- [ ] [Criterion 3]
- [ ] Documentation updated

## Current Status

[Brief summary of where we are. Update this each session.]

## Blockers

[Any blockers. Remove section if none.]

## Quick Links

- [Sessions](sessions/)
- [Decisions](decisions/) (if exists)
- [Plans](plans/) (if exists)
```

---

## Usage Notes

- Keep Goal concise - 2-3 sentences max
- Completion Criteria should be checkable - you know when they're done
- Current Status is the most frequently updated section
- Add/remove sections as needed - this is a starting point
