---

## Usage Notes

- Title should start with "How to" or be clearly action-oriented
- Prerequisites prevent confusion—list them even if obvious
- Steps should be numbered and actionable
- Include verification so readers know it worked
- Common Issues prevents frustration—add issues as you discover them
- Keep guides focused—one task per guide
