# opencode-claude-adapter

Adapter for syncing Claude Code entities (skills, commands, agents, hooks) to OpenCode.

## Overview

This package bridges the gap between Claude Code's entity system and OpenCode, enabling seamless synchronization of:

- Skills
- Commands
- Agents
- Hooks

## Installation

```bash
npm install opencode-claude-adapter
# or
pnpm add opencode-claude-adapter
```
